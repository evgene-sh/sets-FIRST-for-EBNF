# sets FIRST for EBNF
Программа по описанию грамматики, записанному на входном языке в РБНФ, строит множества FIRST для всех нетерминалов грамматики.


## Пример 1
Вход:  

    (E) = (T) [[\+|-] (T)]*.
    (T) = (F) [[\*|/] (F)]*. 
    (T) = (F) [[\*|/] (F)]*. 

Выход:

    {
      '(E)': {'n', '-', '\\('}, 
      '(T)': {'n', '-', '\\('}, 
      '(F)': {'n', '-', '\\('}
    }


## Пример 2
Вход:

    (E) = (T) | a (T).
    (K) = b(K) | eps | (E).
    (T) = [\*|/] (F).
    (F) = n | -(F) | \( (E) \).

Выход:

    {
      '(E)': {'/', 'a', '\\*'}, 
      '(K)': {'/', 'a', 'eps', 'b', '\\*'}, 
      '(T)': {'/', '\\*'}, 
      '(F)': {'n', '\\(', '-'}
    }
